# IMDb Movie Data Analysis

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on a dataset of 1000 highly rated IMDb movies.
The goal is to uncover patterns related to movie popularity, ratings, directors, and revenue.

The analysis includes data cleaning, feature engineering, and multiple visualizations to understand relationships within the dataset.

---

## Dataset

The dataset contains information about 1000 movies, including:

* Title
* Year of release
* Runtime
* Genre
* IMDb rating
* Meta score
* Number of votes
* Gross revenue
* Director
* Main actors

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Key Analyses

The project explores several questions:

* How have movie ratings evolved over time?
* Which directors appear most frequently in top movies?
* Is there a relationship between votes and ratings?
* What genres tend to receive higher ratings?
* What movies generated the highest revenue?

## Key Insights

* Movies with higher vote counts tend to have slightly higher ratings.
* The majority of top-rated movies were released after the year 2000.
* Some directors dominate the dataset with multiple highly rated films.
